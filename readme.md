
## Лабораторная работа №1: Решение систем линейных алгебраических уравнений

### Вариант 11: Метод Гаусса

---

### Описание лабораторной работы

**Цель работы:**
Разработка программы для решения систем линейных алгебраических уравнений (СЛАУ) методом Гаусса, исследование его свойств и сравнение с библиотечными реализациями.

---

### Задание

#### Общие требования:

1. **Номер варианта** определяется по номеру в списке группы согласно ИСУ (вариант 11).

2. **Язык программирования** определяется преподавателем-практиком (Java).

3. **Реализация метода** должна быть выполнена в виде отдельной подпрограммы/метода/класса, в который исходные и выходные данные передаются в качестве параметров.

4. **Размерность матрицы:** n ≤ 20 (задается из файла или с клавиатуры по выбору пользователя).

5. **Ввод данных:** реализовать возможность ввода коэффициентов матрицы как с клавиатуры, так и из файла (по выбору конечного пользователя).

---

### Требования к реализации прямых методов

Для метода Гаусса необходимо реализовать:

- ✓ **Вычисление определителя** по треугольной матрице
- ✓ **Вывод треугольной матрицы** (включая преобразованный столбец B)
- ✓ **Вывод вектора неизвестных:** x₁, x₂, …, xₙ
- ✓ **Вывод вектора невязок:** r₁, r₂, …, rₙ
- ✓ **Сравнение с библиотечным решением:** используя готовые библиотеки, найти решение задачи и определитель. Сравнить полученные результаты и объяснить их сходство/различие в отчёте.

---

### Контрольные вопросы к защите

1. Что является решением системы линейных алгебраических уравнений?
2. Сформулируйте признак отсутствия корней системы линейных алгебраических уравнений.
3. Оцените достоинства и недостатки прямых методов решения СЛАУ.
4. Оцените достоинства и недостатки итерационных методов решения СЛАУ.
5. Как вычислить определитель, если решать СЛАУ методом Гаусса?
6. Идея метода Гаусса с выбором главного элемента по столбцам/строкам?
7. Что такое сходимость итерационного метода?
8. Какое достаточное условие сходимости итерационных методов решения СЛАУ?
9. Чем отличается метод простой итерации от метода Гаусса-Зейделя?
10. Идея метода Гаусса-Зейделя?
11. В каких случаях применяется метод Гаусса с выбором главного элемента?
12. Как определить погрешность решения СЛАУ методом Гаусса?
13. Назовите критерии окончания итерационного процесса метода простой итерации.
14. Если норма преобразованной матрицы будет больше 1, что это означает?
15. Если диагональный элемент равен нулю, можно ли использовать метод Гаусса?
